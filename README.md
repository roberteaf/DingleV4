
# Blooket Bot Backend

A Flask API that uses Playwright to join bots into Blooket games.

## Deploy on Railway

1. Click the "Deploy on Railway" button or manually create a new project from this repo.
2. The service will start automatically.
3. Once deployed, copy your project URL (e.g., `https://your-app.up.railway.app`).
